# 问答
::: warning
问答提问规则：
1. 代码量少的问题可以直接上传截图
2. 代码量大的问题可以把代码上传到git（github、码云等都可以），然后提供git仓库地址
3. 运行错误建议上传完整的错误日志（文本和截图都可以，但要尽量完整），方便定位问题
:::
